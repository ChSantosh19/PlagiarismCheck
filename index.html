How to Use Detectify

Follow these simple steps to check your files for similarities and detect potential plagiarism:

Step 1: Upload Files

Drag and drop multiple files to the upload area or click "browse" to select files. Detectify supports PDFs, Word documents, text files, and programming code files.

Step 2: Process and Compare

Click the "Compare Files" button to run our advanced similarity detection algorithms. The system will extract text, normalize content, and perform thorough comparisons between all file pairs.

Step 3: Review Results

Examine the detailed similarity results in our interactive table. For each file pair, you'll see a similarity percentage and can view side-by-side comparisons with highlighted matching sections.

Key Features

  • Client-side processing - your files never leave your browser
  • Support for multiple file formats (PDF, DOCX, TXT, code files)
  • Advanced similarity detection algorithms
  • Interactive side-by-side comparison with highlighted matches
